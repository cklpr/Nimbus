# Nimbus

Nimbus is a futuristic weather web app built using HTML, CSS, and Vanilla JavaScript. It provides real-time weather data with a cinematic UI, voice narration, ambient sounds, and smart outfit recommendations.

---

# Features

- Real-time weather using the Open-Meteo API
- Current temperature and weather conditions
- Hourly forecast
- 7-day forecast
- Outfit Recommendation system
- Voice weather narration using Web Speech API
- Ambient weather sound effects
- Automatic geolocation
- Manual city search
- Responsive desktop and mobile layout
- Glassmorphism futuristic UI
- Local storage support for saved settings

---

# Outfit Recommendation

Nimbus suggests clothing and accessories based on:
- temperature
- rain probability
- UV index
- wind speed

Examples:
- Wear light clothing
- Bring an umbrella
- Apply sunscreen

---

# Voice Assistant

The app can narrate:
- current weather
- weather conditions
- outfit recommendations

Built using the browser Web Speech API.
